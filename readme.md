# Tab Refresher - Chrome Extension

**All commands are run from project root directory**

### Dev Setup
- `./bin/install.sh`

### Build Project
- `npm run build`

### Incremental Builds
- `npm run dev`

### Create Release/Tag
- `./bin/release.sh`
